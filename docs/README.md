## üöÄ Quick Start

### Installation

```bash
# Build FlowLang from source
git clone https://github.com/yourusername/flowlang.git
cd flowlang
cargo build --release

# Add to PATH
export PATH=$PATH:$(pwd)/target/release
```

### Your First FlowLang Program

```flowlang
-- Hello World in FlowLang
shout("Hello, World!")
```

Run it:
```bash
flowlang run hello.flow
```

### Interactive REPL

```bash
flowlang repl
```

## üéØ Language Features

- **Anime-Themed Syntax**: Keywords inspired by anime (shout, whisper, roar, chant)
- **Epic Error Handling**: ERROR ARC system with Rift, Glitch, VoidTear, Spirit errors
- **Type System**: Essence types (Ember, Silk, Pulse, Flux, Hollow, Constellation, Relic, Spell)
- **Async/Await**: Rituals for asynchronous operations
- **Event Loop**: Node.js-style handle system for long-running processes (timers, servers)
- **Module System**: Import and export with `circle` keyword
- **REPL**: Interactive development environment
- **VS Code Support**: Syntax highlighting and language server

## üåü Philosophy
