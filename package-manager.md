# ğŸŒ¸ **Core Flora Commands (Simple & Essential)**

### **1. Install a module**

```
flora add <module>
```

### **2. Remove a module**

```
flora remove <module>
```

### **3. Update a module**

```
flora update <module>
```

### **4. Update everything**

```
flora upgrade
```

### **5. Show installed modules**

```
flora list
```

### **6. Show module info**

```
flora info <module>
```

### **8. Verify project integrity**

```
flora check
```

### **9. Initialize project dependencies**

```
flora init
```

### **10. Sync dependencies with module.lock**

```
flora sync
```
